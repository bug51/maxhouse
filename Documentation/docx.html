Uploading Via FTP

  • Thank you for your feedback, it's always welcome.Thank you for your feedback, we will consider improving this content.

  • ↑ Back to Top

Follow the steps below to install theme via FTP

Step 1 – Connect/Login to your hosting via FTP software (FileZilla, CuteFTP etc). Once you have downloaded and installed the program you’ll be able to launch the program and proceed with configuring the settings to allow your computer to communicate with your web host account.
Step 2 To configure FileZilla to connect to your website please select  “File” then“Site Manager”.
Step 3. A “Site Manager” window will show up.  Click on the “New Site” button and enter a name for the new connection settings. (You can name it  anything, but best to name with your domain name to keep it straight so that you know where it takes you to).
Step 4.   Now you need to fill in your FTP account information in the box on the right side of the screen.  Please enter “Host name” ( ftp address),  “Server Type” ( FTP),  “User Name and Password” ( your web host account password) which are all available from your webhost.

Step 5. Then click on “Connect” and cross your fingers that you have entered the correct information in the right boxes. If you  entered the information correctly it will connect with your server and populate the area on the right side of the program with folders that shows your website files on the server, ( the left side of the screen displays and gives you access to your local folders on your computer.
Step 6. One of the folders you see is your website main directory.  Typically if your webhost uses a cPanel server, your main root directory is called “public_html”.Your WordPress files should be in that file. The main WordPress files that a beginner needs to work with  is in a folder called “wp-content”. This is the file where you will find you “Plugins” folder as well as your “Themes” folders.
Step 7. To get new plugins or themes added to your WordPress Install through FTP, download zipped files of  plugins and or themes to your computer. When downloaded, extract them to a file on your computer. Drag the unzipped plugin or theme files, individually into either the plugin or the theme file as the case may be.
Step 8. Go to  the WordPress dashboard and go to “Appearance”, “Themes” and see that your new theme now shows up in the list of themes. The downloaded theme should be there. If not refresh, and it should appear ready to be activated. The same with Plugins. The downloaded Plugin should be in the list of “Plugins”and then you must activate it and configure each one as necessary.

Attention! If you face problem in uploading purchased theme, just open a ticket and Chimp Studio team is there to help at Chimp Support.

Setting Directory Types

  • Thank you for your feedback, it's always welcome.Thank you for your feedback, we will consider improving this content.

  • ↑ Back to Top

Directory type is among one of the main feature for DirectoryListing theme. It helps you to set attributes for all your directories. For example, a directory type for Motors allows you to set all features for a post related to motors. When you add motor directory type, you can set it the ads types like free paid, categories for it like car, Jeep, truck, motorcycle etc, make, model & other specific features can be added for a particular directory type. It is important to note there that Directory type is set from admin panel (website back end) and everything added in it will be available for user who wants to post add from front end. to set up Directory type, follow these simple steps;

Step-1. Navigate to admin panel and click on Directory > Directory Type. 
Step-2. Add title & Description for directory type. Scroll to the bottom of page where you will see options to customize your directory type. Select the options that you want to add on your site & hit Publish button. All Done!
Note: All those options you add while creating directory type will be available for users when they will submit an add from front end.

Following options can be availed in directory type settings;

a. Custom Fields
DirectoryListing Theme comes with a builder and you can add custom fields for each of the directory type. When user will be posting a directory from front-end, he will have these custom options or attributes for his directory. Add a feature & you can further customize it by click on it. Every option is self-explanatory. To set custom fields;
Step-1. Navigate to admin panel, click Directory > Directory Type. Add New or edit any existing Directory type. 
Step-2. Scroll to bottom of page and you will see option of Custom fields. Add custom fields of your choice or edit existing ones.  

b. Categories
Categories are sub-directories. For one directory type, there can  be many categories. For example, for motors directory type, there can car, jeep, motorcycle etc. You can select categories to add them on a directory type. To add a new category, here is procedure
Step-1. Navigate to admin panel and click on Category in left panel & click Add New.
Step-2. Add name for category, slug, description & hit Add new Category button in the bottom of page. Category is well added!
Now, to add this category on the directory Type, do the following;
Step-1. Navigate to admin panel, click Directory > Directory Type. Add New or edit any existing Directory type. 
Step-2. Scroll to bottom of page and you will see option of Categories. Select categories that you want to add on Directory type. 

c. Review options
DirectoryListing theme allows users to leave not only an overall rating but individual rating for anything you choose to set up.  You can set up different areas to be rated for different directory types. To add a new review options,
Step-1. Navigate to admin panel, click Directory type. Add New or edit any existing Directory type. 
Step-2. Scroll to bottom of page and you will see Review Options. Select Review option you want to add on Directory type.  

d. Feature list/ Favourites
Each user has its own favourite list where they can add items to watch later. The Favourites feature comes packed with any version of the theme and is a very useful tool for your customers. You can add a list of features which users can add to favourites list later. Also. users in their account area can manage and control the content of the favourite list. To add feature list on your directory type, 
Step-1. Navigate to admin panel, click Directory type. Add New or edit any existing Directory type. 
Step-2. Scroll to bottom of page and you will see option of Feature List. Add as many as you want.

Server Configuration Solution
As with most problems, once you understand the issue, it’s easy to solve it. The server configuration limiting the number of POST variables that can be submitted to the server, we need to increase that.

There are two ways this limit may be imposed
1. PHP’s max_input_vars
The increasingly common issue is the max_input_vars PHP directive. By default, it is set to 1000. To increase this, simply set it in php.ini. I’d suggest 3000 to be safe, but 2000 should be sufficient for most sites.

12 suhosin.post.max_vars = 3000suhosin.request.max_vars = 3000

How do you edit php.ini? That depends on your host. If you have access to the php.ini file, simply add or edit the directive, save, and restart Apache. If you’re with a web host that doesn’t give you access (common with shared hosting), you may have to contact your host and have them make the change for you.
2. Suhosin
Prior to PHP’s max_input_vars, the Suhosin PHP Security module introduced a similar pair of max_vars directives that are identical in purpose. Not all servers run Suhosin, but those that do will need to increase the following directives in php.ini:

Again, you may need to contact your host to get them to make these changes if you don’t know how or can’t do it yourself.
Restarting Apache with the new maximum POST variable values should allow you to submit all of your menu item variables and therefore save new menu items. Hoorah!
Keep in mind this won’t actually recover your lost menu items – that data is gone. The only way to get it back would be to restore from a backup copy of your database.

Adding Languages to WordPress

  • Thank you for your feedback, it's always welcome.Thank you for your feedback, we will consider improving this content.

  • ↑ Back to Top

While U.S English is the default display language for WordPress, the software has the built in capacity to be used in any language. This is achieved by installing .mo languages translation files in a languages folder in wp-content.

If you want to create or modify a .mo file you need to use poedit and the .po file (refer to Translating WordPress for more information).

Installing Languages

  1. Download the required .mo languages translation file from  https://www.debian.org/international/l10n/po/

The link contains both .mo and .po languages translation files. To add a language to your site, you upload the .mo file. The .po files are supplied so you can edit and improve the translations.

  1. Unzip the downloaded files to a folder on your hard drive.
  2. Create a new languages folder on your hard drive and add the desired .mo files to your languages folder
  3.  Launch FileZilla or your FTP client
  4. Locate the languages folder on your computer.
  5. Open up your wp-content folder in your WordPress directory (located within your public_html folder)
  6. Select the new languages folder you created on your computer and drag it into your wp-contentfolder in your WordPress directory.
  7. Dragging the languages folder from your computer into wp-content folder uploads these new languages to your WordPress or WordPress Multisite install however they aren’t enabled.
  8. To enable languages on your site:
  • Open your wp-config.php file in a text editorand search for:

define ('WPLANG', ' ');

  • Edit this line according to the .mofile you’ve just downloaded, e.g. for the Portuguese spoken in Brazil you must add:

define ('WPLANG', 'pt_BR');

Note that if the .mo and .po files don’t exist for a language code called for in wp-config.php then there is no error message, but the code is still used in language_attributes(). This is useful for those of us whose language is similar enough to en_US not to require translation, but who don’t want en-US as the language tag in the blog, instead wanting some other variant of English. For example:

define ('WPLANG', 'en_GB');

  1. Once you have added your language code, save the file.
  2. Upload the modified wp-config.phpfile into the WordPress root directory.
  3. Open your browser and go to your WordPress site. It should now display in the newly-installed language. Along with a list of all added languages in the general settings area

DirectoryListing theme is an outstanding listing theme. It offers users to create their account from front-end without any access or need of site back end. Users can manage their profile, post an ad and keep complete track of their posted ads and payments. This section is help guide for your authors of website on how to use DirectoryListing theme front-end access (you can make it public for front-end users for their ease).

First step to access this amazing option is to login. User can login from front-end via different methods.
Login with Facebook: User can simply click on this option and login with out need of creating profile or filling any form.
Login with Twitter: Just like above option, user can login with his Twitter account.
Login with Google+: Google+ profile have same option. User just need to add his google+ account information and can login.
Sign Up with custom user name: If none of the above option suits user, click sign up option in the bottom of Login window. (This option is opened by default when user visits the login area for the first time). Enter User name and email address in the fields, a confirmation email will be sent to given email address where you can activate the account.

Once the sign up is done, user will have access to all amazing options given in the dashboard.

Note: Admin can make following changes for Login form from website back end;

1- Login form view can be set to Login Window or Drop Down from Appearance > CS Theme Options > Directory Settings > Login View.
2- Login Button Position can be set in Header or Top Strip from Appearance > CS Theme Options > General > Login Button Position.

Create Ad and Listings

With DirectoryListing Theme, user can post as many adds as they want and can manage them very easily. To create an add, following steps must be followed;

Step-1. Login to user account from site front-end and click on Post An Add. (You can also create an add from My Adds section)
Step-2. Set Title & description of the Add, select directory type, fill in all attributes under the directory type and give information in boxes, select the package, select feature add (optional), accept the terms and conditions & click on Create New Directory. Your add is created but it will be subjected to Admin approval for its posting. Once the Admin/moderator approves the add, it will show up on site.

Note: users can manage Adds (edit, delete, renew) in My Adds section of their dashboard. See My Adds section of document for further assistance.

DirectoryListing theme has a complete user management system. My Ads section offers users to see complete record of his ads. Here user can see;
- Ads which he has posted already
- Package value, start date, status (active/expired)
- Renew the package directly with one click
- Edit the ad (for addition or deletion of content, renewal of package etc)
- Delete the ad
- See the number of favourites and likes for the ad

To access this area, just Navigate to site front-end > Login to your Account > Click My Adds.

Favourites section of the user dashboard is the area where all favourite adds of user show up for the record. Any add marked as favourite will directly come in this section, a kind of note book for user add record maintenance.
Adding to favourites: User can add by just clicking favourite button (sometimes a star on the add) on the add.
Removing the favourites: Just navigate to dashboard and click on favourites. Check the star for one by one removal of click Clear all favourites.

Profile is information of user which is available publicly. It can create a strong impression of clients who visit adds. DirectoryListing theme offers full control of profile to user with ease. You can hide your profile from public if you want to. Profile settings area offers following features;
- Upload profile Photo from your photo library up to 1MB size
- Update Avatar of your choice
- Enter detailed description of yourself in About Me section
- Enter CONTACT DETAILS where users/Buyers can contact seller. You can Show/Hide Contact details in profile settings area as well.
- Give Social Media contact information for extended interaction
- Change/Update your PASSWORD
- Enter your PAYPAL information (email address)

To access this area, just Navigate to site front-end > Login to your Account > Click Profile Settings.

Users reviews are necessary part of online business. DirectoryListing theme come pre-wrapped with this option. User can add review to the adds.
How to add Review:
A user can review any add except his own add/s. Just open the add, scroll to the bottom of the page and click on Leave Us a Review, add rating and all done!
Note: User must be logged to add a review.

How to see the Review:
Author of the add can see the reviews given by the users in his dashboard. Just login into dashboard & click on Reviews. All reviews given by the users can be seen here.

NOTE: Only Super Admin of the site has the authority to show/hide reviews of his choice. on secondary user/admin, all added reviews will show up without any difference.
